# WireGuard — منابع و نکات سریع

این فایل حاوی لینک‌ها و خلاصهٔ مفاهیم پایه‌ای است:

- Docs رسمی WireGuard (Quickstart): <https://www.wireguard.com/quickstart/>
- مفاهیم مهم:
  - Key pair: کلید خصوصی (private) و عمومی (public)
  - Peer: هر کلاینت یا سرور یک Peer است
  - Endpoint: آدرس/پورت دسترسی به سرور
  - AllowedIPs: رنج‌هایی که از طریق VPN عبور می‌کنند (مثلاً 0.0.0.0/0 برای کل ترافیک)

**نکته:** برای تولید کلیدها و کانفیگ‌ها از مستندات رسمی یا ابزارهای معتبر استفاده کنید. این فایل صرفاً مرجع مطالعه است.
