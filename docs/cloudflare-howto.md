# Cloudflare Tunnel — منابع و نکات سریع

- Cloudflare Tunnel / Zero Trust docs: <https://developers.cloudflare.com/cloudflare-one/connections/connect-apps/>
- نکات کلی:
  - Tunnel به شما اجازه می‌دهد سرویس را بدون افشای مستقیم IP واقعی به اینترنت متصل کنید.
  - برای نصب نیاز به یک کانفیگ روی سرور/ماشین دارید که باید از مستندات رسمی پیروی شود.
  - از پنل Cloudflare برای مدیریت DNS و احراز هویت استفاده می‌شود.

**هشدار:** حتماً قوانین استفاده Cloudflare و مقررات محلی را بررسی کنید.
